Lizenz
======

Dieses Projekt ist als Hybridmodell organisiert:

- **Hardware-Dokumentation** kann als Open Source bereitgestellt werden.
- **Firmware und bestimmte Softwarekomponenten** können proprietär sein.

Die verbindlichen Lizenzbedingungen sind in den jeweiligen Lizenzdateien im Repository definiert.

Siehe auch: ``LICENSE`` im Projektwurzelverzeichnis.
