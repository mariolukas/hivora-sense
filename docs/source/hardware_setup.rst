Hardware Setup
==============

Dieser Abschnitt beschreibt den mechanischen und elektronischen Aufbau der Hivora-Sense-Station.

3D-Druck (Platzhalter)
----------------------

- Empfohlene Materialien: **PETG** oder **ASA** für Außenanwendungen.
- Platzhalter für Druckprofile (Layerhöhe, Infill, Wandstärke).
- Platzhalter für STL/STEP-Dateien und Bauteilorientierung.

Sensor-Assembly (Platzhalter)
-----------------------------

- Integration der **Wägezelle** (Scale) zur Last-/Aktivitätsmessung.
- Integration des **Temperatursensors** (Temp) zur Umgebungsüberwachung.
- Platzhalter für Verdrahtungsplan und Kalibrierungsablauf.
